Ejemplo de barra de menú en Funcionamiento

En este ejemplo no se pasa ninguna configuración, solo se definen los enlaces del menú:
Para ello hemos usado la llamada del plugin con el siguiente código:


$.leftbar({}, [
['Menú1','./Demos/images/1.jpg','#'],
['Menú2','./Demos/images/2.jpg','#'],
['Menú3','./Demos/images/3.jpg','#'],
['Menú4','./Demos/images/4.jpg','#'],
['Menú5','./Demos/images/5.jpg','#'],
['Menú6','./Demos/images/6.jpg','http://www.fryntiz.es'],
]);
